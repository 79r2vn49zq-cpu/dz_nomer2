# Dependency Visualizer for APT Packages  
Графический инструмент анализа зависимостей для пакетов Ubuntu (формат APT).  
Проект реализован в рамках индивидуального задания (Этапы 1–5).

# Описание проекта

Приложение анализирует зависимости пакетов Ubuntu (APT) и строит:

- полный граф транзитивных зависимостей  
- порядок загрузки (топологическая сортировка)  
- Mermaid-диаграмму графа  

Программа не использует менеджеры пакетов (`apt`, `apt-cache`, `pip`)  
и не использует сторонние библиотеки для получения зависимостей.

---

# Функциональность

Проект включает следующие этапы:

- CLI и параметры конфигурации  
- скачивание и парсинг Packages.gz  
- построение графа зависимостей (DFS без рекурсии)  
- обработка циклов  
- вычисление порядка загрузки  
- вывод графа в формате Mermaid
