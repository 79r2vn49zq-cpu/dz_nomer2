# Описание проекта

Приложение анализирует зависимости пакетов Ubuntu (APT) и строит:

- полный граф транзитивных зависимостей  
- порядок загрузки (топологическая сортировка)  
- Mermaid-диаграмму графа  

Проект **не использует менеджеры пакетов** (`apt`, `apt-cache`, `pip`)  
и **не использует сторонние библиотеки** для получения зависимостей.

---

# Функциональность

### ✔ Этап 1 — CLI и параметры  
- имя пакета  
- URL репозитория  
- режим тестирования  
- максимальная глубина поиска  

### ✔ Этап 2 — получение зависимостей  
- скачивание `Packages.gz`  
- парсинг поля `Depends:`  

### ✔ Этап 3 — граф зависимостей  
- DFS **без рекурсии**  
- обработка циклов  
- ограничение глубины  

### ✔ Этап 4 — порядок загрузки  
- топологическая сортировка  
- детекция циклов  

### ✔ Этап 5 — визуализация  
- генерация диаграмм Mermaid  
- вывод в терминал
